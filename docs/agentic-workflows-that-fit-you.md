# Beyond the Default Stack: Build an Agentic Workflow That Fits You 

AI coding agents are here—but the most effective agentic workflows are not built by accident. In this workshop, discover how to create an intentional agentic development approach. You’ll learn how to define the constraints, context, and collaboration patterns that help AI produce code that is cleaner, more maintainable, and better aligned to the way you work.

This workshop goes beyond prompt tips to focus on the mindset and mechanics of durable human-AI collaboration. Participants will explore how to guide AI behavior, adapt to friction points, and ask the right questions continuously to improve code quality over time. Join us to see how agentic coding can become a flexible, personalized development practice—without being tied to a single toolchain.
